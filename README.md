# Personal website for GitHub Pages

## Run locally
Open `index.html` in your browser.

## Deploy with GitHub Pages
1. Create a public repository named `<your-github-username>.github.io`
2. Upload all files from this folder to the repository root
3. Commit and push to `main`
4. In GitHub, go to Settings → Pages
5. Set Source to `Deploy from a branch`
6. Select `main` and `/ (root)`
7. Your website will appear at `https://<your-github-username>.github.io/`

## Optional updates
- Replace `assets/profile.png` with a higher-resolution portrait
- Update the Google Scholar link in `index.html`
- Add paper, project, or code links beneath each publication
